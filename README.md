# 숫자 야구 게임
## 기능 구현
- ###Ball
  - [ ] Ball과 Ball을 비교하여 Strike, Ball, Nothing 확인 기능
- ###Balls
  - [ ] 각 Ball을 확인 후, 결과 값을 반환하는 기능
- ###Result
  - [ ] Strike, Ball 결과 값 저장
  - [ ] 결과 값 format 출력 기능
- ###기타 로직
  - [ ] 사용자 입력 값 validation 기능
- ### 게임
  - [ ] 실행 기능
  

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
